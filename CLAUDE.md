# Oricade — Claude Code Guide

Browser-based, multi-level 3D arcade game. Static site, no backend. See `SPEC.md` for the full design.

## Git

- **Main branch:** `master`
- **Stash:** always `git stash apply`, never `git stash pop`
- **Issue branches:** `fix/issue-{N}-{slug}` (bug) · `feat/issue-{N}-{slug}` (feature)

## Stack

- Three.js (WebGL), no backend, deployed as a static bundle.
- See `SPEC.md` — Technical Stack for details.
